# Project 27: Media Archiver

## Overview
A media acquisition utility built to download high-quality video and audio assets from web platforms. 

## Features
- **Multi-Format Support:** Option to extract raw video or convert to high-bitrate MP3.
- **Dynamic Naming:** Automatically names files based on the source metadata.
- **Powered by yt-dlp:** Industry-standard backend for reliable extraction.

## Usage
1. Run `python archiver.py`
2. Enter the URL of the media you wish to archive.
3. Select format (MP4 or MP3).
