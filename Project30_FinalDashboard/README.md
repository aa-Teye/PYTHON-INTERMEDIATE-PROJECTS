# Project 30: Sentinel Executive Dashboard

## Overview
The capstone project of the 30-Day Python Challenge. A full-stack web dashboard designed for system monitoring and business intelligence.

## Features
- **Interactive UI:** Built with Streamlit for seamless navigation.
- **Hardware Telemetry:** Real-time CPU, RAM, and Disk monitoring.
- **Data Analytics:** Visual representation of business sales and revenue.
