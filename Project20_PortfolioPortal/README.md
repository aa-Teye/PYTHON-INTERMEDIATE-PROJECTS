# Project 20: Master Portfolio Portal

## Overview
A centralized web dashboard developed using Flask to aggregate and showcase Python projects. This portal serves as a unified interface for system utilities, CRUD applications, and network monitors.

## Features
- Dynamic Routing: Automatically renders project cards using Jinja2 templates.
- Responsive Design: Built with Bootstrap 5 for compatibility across desktop and mobile.
- Scalable Architecture: Designed to easily integrate all 30 projects in the series.

## Execution
1. Navigate to Project20_PortfolioPortal.
2. Run the server:
   python app.py
3. View the dashboard at http://127.0.0.1:5000
