## Description
NetWatcher is a network monitoring tool designed to track the connectivity and latency of various web services. It logs real-time performance data into a CSV file for long-term health tracking of critical URLs.

## Features
* **Latency Tracking**: Measures server response time in milliseconds.
* **Status Logging**: Automatically records uptime and downtime events.
* **CSV Export**: Generates structured data for performance analysis.

## Setup
1. Install dependencies: `pip install requests`
2. Run the monitor: `python watcher.py`
