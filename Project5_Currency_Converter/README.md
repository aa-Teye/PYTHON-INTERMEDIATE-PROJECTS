## Description
This project is an intermediate-level tool that fetches real-time exchange rates using the **ExchangeRate-API**. It demonstrates my ability to handle live data, parse JSON objects, and manage external API requests.

## Key Features
* **Live API Integration**: Fetches current global rates.
* **JSON Parsing**: Extracts specific rate data from complex nested dictionaries.
* **Input Validation**: Handles user currency codes and numerical amounts.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the script: `python converter.py`
