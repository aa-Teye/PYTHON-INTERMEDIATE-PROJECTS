# Project 24: Cyber-Escape (Text Adventure Game)

## Overview
A logic-based text adventure game that simulates a system breach. This project demonstrates state management and recursive function calls in Python.

## Technical Features
- **Dynamic Inventory:** Tracks player items to unlock specific game branches.
- **Recursive Navigation:** Uses modular functions to represent different game environments.
- **Input Parsing:** Sanitizes user commands to drive game progression.

## Execution
1. Run: `python game.py`
2. Follow the on-screen prompts to escape the vault.
