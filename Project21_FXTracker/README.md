# Project 21: Live FX Tracker

## Overview
A financial utility that interfaces with public Exchange Rate APIs to provide real-time currency conversion. This project demonstrates API consumption and JSON data parsing in Python.

## Technical Features
- **Live Data Fetching:** Uses the `requests` library to perform HTTP GET requests.
- **Dynamic Conversion:** Handles user-defined currency pairs (USD, GHS, EUR, etc.).
- **Error Handling:** Robust checks for network timeouts and invalid currency codes.

## Installation
1. Navigate to Project21_FXTracker.
2. Install dependencies: `pip install requests`
3. Run: `python tracker.py`
