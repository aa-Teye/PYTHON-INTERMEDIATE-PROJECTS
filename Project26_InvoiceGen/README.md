# Project 26: Automated Invoice Generator

## Overview
A business automation tool designed for **Dominion Tech Solutions** to generate professional PDF receipts for laptop sales. 

## Features
- **Branded Header:** Automatically applies business branding and slogans.
- **Auto-Date:** Fetches current system time for transaction records.
- **PDF Export:** Generates clean, printable documents using the `fpdf2` library.

## Usage
1. Install dependencies: `pip install -r requirements.txt`
2. Run `python generator.py`
3. Enter sale details to produce a PDF invoice in the project folder.
