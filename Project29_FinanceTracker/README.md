# Project 29: Sentinel Finance Ledger

## Overview
A database-driven personal finance utility designed for income/expense tracking and budgetary analysis. 

## Technical Stack
- **Database:** SQLite3 (Serverless SQL engine).
- **Data Analysis:** Aggregate SQL queries for balance calculation.
- **Visualization:** Matplotlib pie charts for categorical spending analysis.

## Usage
1. Run `python tracker.py`
2. Input your transactions (Laptop sales, Church offerings, personal spending).
3. Use the 'Summary' option to view your current liquidity.
